Virtualization: The ability to increase computing efficiency



Virtualization is a component within Cloud Computing used to separate the Operating System from the underlying hardware. This allows you to overcome the limitations of physical hardware by creating multiple server instances on one physical server. This has multiple benefits, including:

- Data segregation

- Greater flexibility and resource utilization

- Efficiency gains in processing power, and

- Reduced electrical costs.

Virtualization is the backbone of Cloud Computing, allowing people to deploy their web applications or services completely off-premise in a managed, hosted environment.
